Database Normalization Summary
  • 0NF: Single table with all data, including repeating groups
  • 1NF: Separated repeating groups into different tables
  • 2NF: Eliminated partial dependencies by creating separate entities
  • 3NF: Eliminated transitive dependencies by normalizing address information
